# earthquakePulseMap Repository

## Overview
Real-time earthquake monitoring and visualization system.

## Contents
- .git/: Git repository files
- .gitignore: Git ignore file
- README.md: Documentation
- biome.json: Biome configuration
- branding/: Branding files
- dist/: Distribution files
- index.html: Main HTML file
- node_modules/: Node.js modules
- orignal/: Original files
- package-lock.json: Package lock file
- package.json: Package configuration
- public/: Public assets
- scripts/: Scripts directory
- src/: Source code
- tsconfig.app.json: TypeScript app configuration
- tsconfig.json: TypeScript configuration
- tsconfig.node.json: TypeScript node configuration
- vite.config.ts: Vite build configuration

## Purpose
This repository implements a real-time earthquake monitoring and visualization system. It provides tools for tracking seismic activity and visualizing earthquake data in real-time for monitoring and analysis purposes.
